highlight

open var highlight: UiComponent?

Overrides Focusable.highlight

A UiComponent for rendering the focus highlight is passed to components where the focus manager requests a highlight. The focus highlighted component is then responsible for adding the highlight to the display graph and sizing / positioning the component in its updateLayout, and rendering the component in its render.
