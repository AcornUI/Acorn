getChildUnderPoint

fun UiComponentRo.getChildUnderPoint(canvasX: Float, canvasY: Float, onlyInteractive: Boolean): UiComponentRo?

Given a global position, casts a ray in the direction of the camera, returning the deepest enabled interactive element at that position. If there are multiple objects at this position, only the top-most object is returned. (by child index, not z value)
