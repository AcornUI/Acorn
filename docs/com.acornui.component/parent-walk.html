parentWalk

inline fun UiComponentRo.parentWalk(callback: (UiComponentRo) -> Boolean): UiComponentRo?

Traverses this ChildRo's ancestry, invoking a callback on each parent up the chain. (including this object)

Parameters

callback - The callback to invoke on each ancestor. If this callback returns true, iteration will continue, if it returns false, iteration will be halted.

Return
If callback returned false, this method returns the element on which the iteration halted.
