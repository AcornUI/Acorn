onElementAdded

protected open fun onElementAdded(index: Int, element: UiComponent): Unit

Overrides ElementContainerImpl.onElementAdded

Invoked when an external element has been added. If this is overriden and the addChild is delegated, the onElementRemoved should mirror the delegation.

Example:

private val otherContainer = addChild(container())

override fun onElementAdded(index: Int, child: T) { otherContainer.addElement(child) } override fun onElementRemoved(index: Int, child: T) { otherContainer.removeElement(child) }
