UiComponentImpl(owner: Owned, native: NativeComponent = owner.inject(NativeComponent.FACTORY_KEY)(owner))

The base for every AcornUi component. UiComponent provides lifecycle, validation, interactivity, transformation, and layout.

Parameters

owner - The creator of this component. This is used for dependency injection, style inheritance, and when the owner has been disposed, this component will then be disposed.

native - Some back-ends (like DomApplication) will create native components that the acorn display hierarchy controls.
