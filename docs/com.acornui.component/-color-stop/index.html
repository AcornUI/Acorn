percent

 var percent: Float?

The percent towards the end position of the gradient. This number is absolute, not relative to the previous color stop. If neither pixels or percent is set, this stop will be halfway between the previous stop and the next.

pixels

 var pixels: Float?

The number of pixels towards the end position of the gradient. This number is absolute, not relative to the previous color stop. If neither pixels or percent is set, this stop will be halfway between the previous stop and the next.
