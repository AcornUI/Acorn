Overrides UiComponentImpl.getChildrenUnderPoint

Overrides UiComponentRo.getChildrenUnderPoint

Given a screen position, casts a ray in the direction of the camera, populating the out list with the components that intersect the ray.

Parameters

canvasX - The x coordinate relative to the canvas.

canvasY - The y coordinate relative to the canvas.

onlyInteractive - If true, only elements whose interactivity is enabled will be returned.

returnAll - If true, all intersecting elements will be added to out, if false, only the top-most element. The top-most element is determined by child index, not z value.

out - The array list to populate with elements.

rayCache - If the ray is already calculated, pass this to avoid re-calculating the pick ray from the camera.
