focusOrder

open var focusOrder: Float

Overrides Focusable.focusOrder

The focus order weight. A higher number means this component will be later in the order. (An order of 1f will be focused before a Focusable component with the same parent with an order of 2f) In the case of a tie, the order within the display graph (breadth-first) is used.
