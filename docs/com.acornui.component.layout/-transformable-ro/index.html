concatenatedTransform

 abstract val concatenatedTransform: Matrix4Ro

The global transform of this component, of all ancestor transforms multiplied together. Do not modify this matrix directly, it will be overwritten on a TRANSFORM validation.

concatenatedTransformInv

 abstract val concatenatedTransformInv: Matrix4Ro

Returns the inverse concatenated transformation matrix. Note that this is a heavy operation and should be used judiciously. It is not part of the normal validation cycle.

customTransform

 abstract val customTransform: Matrix4Ro?

If this is not null, this custom transformation matrix will be used. Note that if this is set, all properties that would otherwise generate the transformation matrix are no longer applicable. scaleX, scaleY, scaleZ, rotationY, rotation, originX, originY, originZ
