bound

fun bound(sizeConstraints: SizeConstraintsRo): Unit

Bound the width and height by another size constraints object. This will result in the following adjustment: width.min = max(widthA, widthB) height.min = max(heightA, heightB) width.max = min(widthA, widthB) height.max = min(heightA, heightB)
