SequencedLayout

interface SequencedLayout<in S, out T : LayoutData> : LayoutAlgorithm<S, T>

A sequenced layout is a layout where the the elements are laid out in a serial manner. This means that for elements {a, b, c}, b will always be spatially positioned between a and c.
