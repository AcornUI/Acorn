ParentRo

 interface ParentRo<out T> : ChildRo

An abstract parent to children relationship of a parameterized type. Acorn hierarchies are conventionally A : B -> B Where B is a child of A, and A is a sub-type of B. This means that when using recursion throughout the tree, all nodes can be considered to be a ChildRo of type B, but it must be type checked against a ParentRo of type A before continuing the traversal.
