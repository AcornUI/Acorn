findChildPreOrder

inline fun <reified T> T.findChildPreOrder(callback: (T) -> Boolean, reversed: Boolean): T?

Given a callback that returns true if the descendant is found, this method will return the first descendant with the matching condition. The tree traversal will be pre-order.
