inline fun <reified T> T.childWalkLevelOrder(callback: (T) -> TreeWalk, reversed: Boolean): Unit

A level-order child walk. Traverses this parent's hierarchy from top to bottom, breadth first, invoking a callback on each ChildRo element (including this receiver object).

Parameters

reversed - If true, the last child will be added to the queue first.

callback - The callback to invoke on each child.
