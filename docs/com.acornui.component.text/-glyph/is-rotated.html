isRotated

val isRotated: Boolean

If not rotated, the uv coordinates are: Top-left: (u, v), Top-right: (u2, v), Bottom-right: (u2, v2), Bottom-left: (u, v2) If rotated, the uv coordinates are: Top-left: (u2, v), Top-right: (u2, v2), Bottom-right: (u, v2), Bottom-left: (u, v)
