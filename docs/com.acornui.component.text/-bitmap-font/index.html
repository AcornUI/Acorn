getGlyphSafe

 fun getGlyphSafe(char: Char): Glyph

Returns the glyph associated with the provided char. If the glyph is not found, if the char is whitespace, an empty glyph will be returned. If the glyph is not found and is not whitespace, a missing glyph placeholder will be returned.
