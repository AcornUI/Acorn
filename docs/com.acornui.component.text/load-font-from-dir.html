fun Owned.loadFontFromDir(fontStyle: FontStyleRo, fntPath: String, imagesDir: String): Deferred<BitmapFont>

Loads a font where the images are standalone files in the specified directory.

Parameters

fntPath - The path of the font data file. (By default the font data loader expects a .fnt file in the AngelCode format, but this can be changed by associating a different type of loader for the BitmapFontData asset type.)

imagesDir - The directory of images
