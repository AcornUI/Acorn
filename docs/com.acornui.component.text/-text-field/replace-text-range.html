replaceTextRange

open fun replaceTextRange(startIndex: Int, endIndex: Int, newText: String): Unit

Replaces the given range with the provided text. This is functionally the same as: text.substring(0, startIndex) + newText + text.substring(endIndex, text.length)

Parameters

startIndex - The starting character index for the replacement. (Inclusive)

endIndex -

The ending character index for the replacement. (Exclusive)

E.g. +text("Hello World") { replaceTextRange(1, 5, "i") // Hi World }
