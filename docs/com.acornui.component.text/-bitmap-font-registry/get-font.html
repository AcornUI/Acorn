fun getFont(fontStyle: FontStyleRo, warnOnNotFound: Boolean = true): Deferred<BitmapFont>?

Parameters

fontStyle - The style used as a key via register

warnOnNotFound - If warnOnNotFound is true and the font style was never registered, a logger warning will be issued. This warning will only occur once per font style.
