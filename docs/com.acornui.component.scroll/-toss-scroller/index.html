Properties

dampening

 var dampening: Float

Dampening affects how quickly the toss velocity will slow to a stop. Make this number 0 < dampening < 1. Where 1 will go forever, and 0 will prevent any momentum.

dragAttachment

 val dragAttachment: DragAttachment

enabled

 var enabled: Boolean

Enables or disables this toss scroller. If this is set to false in the middle of a toss, the toss will still finish as if the drag were immediately released. Use stop to halt the current velocity.
