updateLayoutEntry

abstract fun updateLayoutEntry(explicitWidth: Float?, explicitHeight: Float?, element: LayoutElement, currentIndex: Int, startIndex: Float, lastIndex: Int, previousElement: LayoutElement?, isReversed: Boolean, props: S): Unit

Sizes and positions the given layout element.

Parameters

explicitWidth - The explicit width.

explicitHeight - The explicit height.

element - The layout element to size and position.

currentIndex - The index of the entry to lay out.

startIndex - The index of the first virtualized item.

lastIndex - The index of the last item.

previousElement - The layout element previously updated. May be null if this is the first item.

isReversed - If true, the layout is iterating in reverse order.

props - The style parameters object.

Return
out Returns x,y coordinates representing the measured width and height.
