SignalBase

abstract class SignalBase<T : Any> : Signal<T>, Disposable

Signals are a way to implement an observer relationship for a single event.

The expected concurrent behavior for a handler adding or removing is as follows:

If a handler is added within a handler, the new handler will be invoked in the current iteration. If a handler is removed within a handler, the removed handler will not be invoked in the current iteration.
