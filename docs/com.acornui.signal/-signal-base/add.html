add

open fun add(handler: T, isOnce: Boolean): Unit

Overrides Signal.add

Adds a handler to this signal.

If this signal is currently dispatching, the added handler will not be dispatched until the next dispatch call.

Parameters

handler - The callback that will be invoked when dispatch() is called on the signal.

isOnce - A flag, where if true, will cause the handler to be removed immediately after the next dispatch.
