StyleTag

interface StyleTag : StyleFilter

Style tags are markers placed on the display hierarchy that are used for filtering which styles from StyleRule objects are applied.

A StyleTag may be used as a StyleFilter, with the default implementation passing the filter if the current target contains this tag.
