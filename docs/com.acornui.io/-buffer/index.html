Buffer

interface Buffer

An interface common to ReadBuffer and WriteBuffer These methods manipulate the getPosition and mark cursors.

Properties

capacity

 abstract val capacity: Int

The capacity of this buffer. This typically will never change.

hasRemaining

 abstract val hasRemaining: Boolean

Indicates if there are elements remaining in this buffer, that is if {@code position < limit}.

limit

 abstract val limit: Int

Returns the limit of this buffer.

position

 abstract var position: Int

The position of this buffer. If the mark is set and it is greater than the new position, then it is cleared. Must be not negative and not greater than limit.

remaining

 open val remaining: Int

Returns the number of remaining elements in this buffer, that is {@code limit - position}.

Functions

flip

 abstract fun flip(): Buffer

Flips this buffer.

mark

 abstract fun mark(): Buffer

Marks the current position, so that the position may return to this point later by calling reset().

reset

 abstract fun reset(): Buffer

Resets the position of this buffer to the mark.

rewind

 abstract fun rewind(): Buffer

Rewinds this buffer.
