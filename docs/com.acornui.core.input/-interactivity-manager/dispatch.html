abstract fun dispatch(target: UiComponentRo, event: InteractionEvent, useCapture: Boolean = true, useBubble: Boolean = true): Unit

Dispatches an interaction for a single interactive element. This will first dispatch a capture event from the stage down to the given target, and then a bubbling event up to the stage.

Parameters

target - The target for the event.

event - The event to dispatch on each ancestor.

useCapture - If true, there will be a capture phase. That is, starting from the highest ancestor (the Stage if the target is active) the event will be dispatched on each ancestor down to (and including) the target.

useBubble - If true, there will be a bubble phase. That is, starting from the target, each ancestor will have the event dispatched. If both useCapture and useBubble are false, only the target will have the event dispatched. (Using the bubble-phase signal)
