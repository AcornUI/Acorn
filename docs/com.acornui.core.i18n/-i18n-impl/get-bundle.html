fun getBundle(locales: List<Locale>, bundleName: String): I18nBundle

Overrides I18n.getBundle

Returns a localization bundle for the given bundle name and locales. Use this if you wish to get localization values for a specific locale chain, as opposed to the current locale.

Parameters
