cellClicked

val cellClicked: Signal<(CellLocation<E>, Cancel) -> Unit>

Dispatched when the contents area has been clicked. Use CellLocation.isValid to determine whether or not a valid cell has been clicked before getting the row/column. If the cancel object is invoked, the default behavior for clicking a cell will be prevented. (I.e. cell editing)
