sourceIndexToLocal

fun sourceIndexToLocal(sourceIndex: Int): RowLocation<E>?

Given an index within the data list, this will return a RowLocation object representing the local position which takes into account filtering, sorting, and grouping. This will return null if the element is filtered out.

Note - If the groups do not provide unique filtering, this method returns the first group found where the element is not filtered out.

Parameters

sourceIndex - The index of the element within data to convert. This should be between 0 and data.lastIndex (inclusive)
