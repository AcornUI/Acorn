iterateVisibleColumns

fun iterateVisibleColumns(callback: (columnIndex: Int, column: DataGridColumn<E, *>, columnX: Float, columnWidth: Float) -> Boolean): Unit

Iterates over the currently visible columns. The callback will be invoked for every column fully or partially visible, providing the the columnIndex, column, x position, and width.
