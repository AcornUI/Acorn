headerCellHAlign

var headerCellHAlign: HAlign

The horizontal alignment of the header cells. Note that this does not affect text alignment. To change the cell's text alignment, set the text field's flowStyle horizontalAlign property. This can be done universally by setting a style rule for DataGrid.HEADER_CELL
