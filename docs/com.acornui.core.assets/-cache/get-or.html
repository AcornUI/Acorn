getOr

open fun <T : Any> getOr(key: CacheKey<T>, factory: () -> T): T

Retrieves the cache value for the given key if it exists. Otherwise, constructs a new value via the factory and adds the new value to the cache.

Parameters

key - The key to use for the cache index.

Return
Returns the cached value.
