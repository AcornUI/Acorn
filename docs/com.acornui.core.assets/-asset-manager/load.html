load

abstract fun <T> load(path: String, type: AssetType<T>): AssetLoader<T>

Loads an asset. To cancel the loading, call cancel on the return value. To await the result, create a coroutine, and call 'await'.

launch {
   val a = load("foo.txt", AssetType.TEXT)
   val b = load("bar.png", AssetType.TEXTURE)
   val str = a.await()
   val texture = b.await()
}

Parameters

path - The location of the asset.

type - The type of asset the path represents. For common asset types, see AssetType
