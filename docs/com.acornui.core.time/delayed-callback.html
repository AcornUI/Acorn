delayedCallback

fun Scoped.delayedCallback(duration: Float, callback: () -> Unit): CallbackWrapper

Returns a wrapper that when invoked, will call callback after duration seconds. Every consecutive invocation of the wrapper will reset this timer. Useful for things like auto-save after the user is done typing for a set amount of time.

Parameters

duration - The number of seconds before the callback is invoked.

callback - The function to call after duration seconds.
