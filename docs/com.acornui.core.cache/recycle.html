Recycles a list of item renderers, creating or disposing renderers only as needed.

Parameters

data - The updated list of data items.

existingElements - The stale list of item renderers. This will be modified to reflect the new item renderers.

factory - Used to create new item renderers as needed. configure will be called after factory to configure the new element.

configure - Used to configure the element.

disposer - Used to dispose the element.

equality - If set, uses custom equality rules. This guides how to know whether an item can be recycled or not.
