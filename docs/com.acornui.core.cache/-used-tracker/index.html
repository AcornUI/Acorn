UsedTracker

class UsedTracker<E>

Similar to IndexedCache, to use this class, make a series of markUsed calls to indicate that an element was ued, then call forEach to iterate over the elements that were marked last set, but weren't marked this set, then call flip to set the used elements as unused for the next set.
