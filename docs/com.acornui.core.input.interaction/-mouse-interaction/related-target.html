relatedTarget

open var relatedTarget: UiComponentRo?

Overrides MouseInteractionRo.relatedTarget

On a mouse out interaction, the relatedTarget will be the new over target (or null if there isn't one) On a mouse over interaction, the relatedTarget will be the previous over target (or null if there wasn't one)
