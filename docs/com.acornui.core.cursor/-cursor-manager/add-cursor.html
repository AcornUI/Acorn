addCursor

abstract fun addCursor(cursor: Cursor, priority: Float = 0f): CursorReference

Adds a cursor at the given index in the stack. (Only the last cursor will be displayed.)

Parameters

cursor - The cursor to add. Use StandardCursors to grab a cursor object.

priority - The priority of the cursor. Use CursorPriority to get useful default priorities.

Return
Returns the cursor reference of the cursor added, this can be used to remove the cursor.
