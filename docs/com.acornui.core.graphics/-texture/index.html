refDec

 abstract fun refDec(): Unit

Decrements the number of places this Texture is used. If the count reaches zero, the texture will be unloaded from the gpu.

refInc

 abstract fun refInc(): Unit

Increments the number of places this Texture is used. If this Texture was previously not referenced, this texture will be initialized and uploaded to the GPU.
