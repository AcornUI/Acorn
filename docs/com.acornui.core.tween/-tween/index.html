Tween

interface Tween

A Tween is an object representing an interpolation over time.

Properties

allowCompletion

 abstract var allowCompletion: Boolean

If this is true, completed will be dispatched when the play head scrubs past an endpoint regardless if looping is enabled.

alpha

 open var alpha: Float

The current 0-1 progress of the tween.

completed

 abstract val completed: Signal<(Tween) -> Unit>

Dispatched when the play head has scrubbed past a boundary, either 0f or duration. If looping is enabled, this will not be invoked unless allowCompletion is called.

currentTime

 abstract var currentTime: Float

The current time of the tween, in seconds.

duration

 abstract val duration: Float

The total duration of this tween, in seconds.

durationInv

 abstract val durationInv: Float

1f / duration.

loopAfter

 abstract var loopAfter: Boolean

If true, when this tween's currentTime >= duration, it will loop back to zero.

loopBefore

 abstract var loopBefore: Boolean

If true, when this tween's currentTime <= 0f, it will loop back to duration.

paused

 abstract var paused: Boolean

If true, update will not advance the current time. This does not affect directly changing the current time.

startTime

 abstract var startTime: Float

When the tween is rewound using rewind, the currentTime will be set to this value. This can be set to a negative value to indicate a delay.
