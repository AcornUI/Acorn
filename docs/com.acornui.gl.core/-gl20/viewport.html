viewport

abstract fun viewport(x: Int, y: Int, width: Int, height: Int): Unit

Specifies the affine transformation of x and y from normalized device coordinates to window coordinates.

Note - if the current frame buffer is the window, the width and height should be multiplied by com.acornui.core.graphics.Window.scaleX and com.acornui.core.graphics.Window.scaleY

Parameters

x - Specify the lower left corner of the viewport rectangle, in pixels. The initial value is (0,0).

y -

width - Specify the width and height of the viewport. When a GL context is first attached to a window, width and height are set to the dimensions of that window.
