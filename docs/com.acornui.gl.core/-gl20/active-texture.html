activeTexture

abstract fun activeTexture(texture: Int): Unit

Set the texture unit subsequent texture operations apply to.

must be one of TEXTURE0, TEXTURE1, to getParameter(gl.MAX_COMBINED_TEXTURE_IMAGE_UNITS) - 1. The default value is TEXTURE0. A texture must be bound to the active texture unit using bindTexture().
