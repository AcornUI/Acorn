drawElements

abstract fun drawElements(mode: Int, count: Int, type: Int, offset: Int): Unit

Draws primitives using the vertex buffer data (stored in the ARRAY_BUFFER buffer) and the index buffer data (stored in the ELEMENT_ARRAY_BUFFER buffer).

Parameters

mode - must be one of POINTS, LINE_STRIP, LINE_LOOP, LINES, TRIANGLE_STRIP, TRIANGLE_FAN, or TRIANGLES.

count - the number of elements to be rendered.

type - must be one of UNSIGNED_BYTE or UNSIGNED_SHORT.

offset - an offset in the element array buffer. Must be a valid multiple of the size of the given type. You must call enableVertexAttribArray() for each attribute in the vertex shader that uses the vertex data.
