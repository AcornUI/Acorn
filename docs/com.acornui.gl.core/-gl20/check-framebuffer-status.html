checkFramebufferStatus

abstract fun checkFramebufferStatus(target: Int): Int

Returns one of the following to indicate the current status of the framebuffer: FRAMEBUFFER_COMPLETE, FRAMEBUFFER_INCOMPLETE_ATTACHMENT, FRAMEBUFFER_INCOMPLETE_DIMENSIONS, FRAMEBUFFER_INCOMPLETE_MISSING_ATTACHMENT, or FRAMEBUFFER_UNSUPPORTED.
