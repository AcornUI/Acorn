abstract fun readPixels(x: Int, y: Int, width: Int, height: Int, format: Int, type: Int, pixels: NativeBuffer<Byte>): Unit

Reads pixels from the framebuffer and copies them to data. Note: If the window is obscured and the frame buffer being read is not an off-screen buffer, the data may contain garbage.
