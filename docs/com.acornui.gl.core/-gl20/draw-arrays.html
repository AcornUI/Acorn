drawArrays

abstract fun drawArrays(mode: Int, first: Int, count: Int): Unit

Draws primitives using the vertex buffer data (stored in the ARRAY_BUFFER buffer).

Parameters

mode - must be one of POINTS, LINE_STRIP, LINE_LOOP, LINES, TRIANGLE_STRIP, TRIANGLE_FAN, or TRIANGLES.

first - specifies the index of the first vertex to draw.

count - specifies the number of vertices to draw. You must call enableVertexAttribArray() for each attribute in the vertex shader that uses the vertex data.
