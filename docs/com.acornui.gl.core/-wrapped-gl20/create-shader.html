createShader

open fun createShader(type: Int): GlShaderRef

Overrides Gl20.createShader

Creates a vertex or fragment shader. type must be either VERTEX_SHADER or FRAGMENT_SHADER. Shaders must be compiled using compileShader() and then attached to a GlProgram using attachShader() before they can be used.
