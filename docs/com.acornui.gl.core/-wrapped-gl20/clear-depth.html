clearDepth

open fun clearDepth(depth: Float): Unit

Overrides Gl20.clearDepth

Specifies the value to fill the depth buffer when reset() is called with the DEPTH_BUFFER_BIT. depth is clamped to the range 0 (near) to 1 (far). Defaults to 1 if not specified.
