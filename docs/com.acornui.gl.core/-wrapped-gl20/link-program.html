linkProgram

open fun linkProgram(program: GlProgramRef): Unit

Overrides Gl20.linkProgram

linkProgram links the program object specified by program. A shader object of type GL_VERTEX_SHADER attached to program is used to create an executable that will run on the programmable vertex processor. A shader object of type GL_FRAGMENT_SHADER attached to program is used to create an executable that will run on the programmable fragment processor.
