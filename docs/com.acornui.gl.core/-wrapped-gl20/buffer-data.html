bufferData

open fun bufferData(target: Int, size: Int, usage: Int): Unit

Overrides Gl20.bufferData

Creates the storage for the currently bound buffer.

Parameters

target - must be one of ARRAY_BUFFER or ELEMENT_ARRAY_BUFFER.

size - the size in bytes of the buffer to allocate.

usage - must be one of STREAM_DRAW, STATIC_DRAW, or DYNAMIC_DRAW.
