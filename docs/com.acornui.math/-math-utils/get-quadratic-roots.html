getQuadraticRoots

fun getQuadraticRoots(a: Float, b: Float, c: Float, out: MutableList<Float>): Unit

Given a quadratic equation of the form: y = ax^2 + bx + c, returns the solutions to x where y == 0f Uses the quadratic formula: x = (-b += sqrt(b^2 - 4ac)) / 2a

Parameters

a - The a coefficient.

b - The b coefficient.

c - The c coefficient.

out - The list to populate with the solutions.

Return
Returns the x values where y == 0f. This may have 0, 1, or 2 values.
