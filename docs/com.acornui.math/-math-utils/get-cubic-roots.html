Given a cubic equation of the form: y = ax^3 + bx^2 + cx + d, returns the solutions to x where y == 0f

Parameters

a - The a coefficient.

b - The b coefficient.

c - The c coefficient.

c - The d coefficient.

out - The list to populate with the solutions.

Return
Returns the x values where y == 0f. This may have 0, 1, 2 or 3 values.
