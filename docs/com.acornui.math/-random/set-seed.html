fun setSeed(seed: Long): Unit

Sets the internal seed of this generator based on the given {@code long} value.

The given seed is passed twice through an hash function. This way, if the user passes a small value we avoid the short irregular transient associated with states having a very small number of bits set.

Parameters

seed - a nonzero seed for this generator (if zero, the generator will be seeded with {@link Long#MIN_VALUE}).
