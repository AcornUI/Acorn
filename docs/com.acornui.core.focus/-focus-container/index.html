FocusContainer

interface FocusContainer : LayoutElementRo

A demarcation for focus order. That is, focus order is considered to be relative to the closest FocusContainer ancestor.

A focus container does not need to be focusable, although it can be.

Properties

focusOrder

 abstract var focusOrder: Float

The focus order weight. A higher number means descendant focusable elements will be focused later in the order than focusable elements descendant of a sibling focus container with a lower focus order.
