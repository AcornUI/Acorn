open fun focused(): Focusable?

Overrides FocusManager.focused

Returns the currently focused element. This will be null if the root of the focus manager is being removed.

open fun focused(value: Focusable?): Unit

Overrides FocusManager.focused

Sets the currently focused element. The provided focusable element may have Focusable.focusEnabled set to false; It will still be focused.

Parameters

value - The target to focus. If this is null, the manager's root will be focused. (This is typically the stage)
