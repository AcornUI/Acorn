ConcurrentListIterator

interface ConcurrentListIterator<out E> : Clearable, ListIterator<E>, Iterable<E>, Disposable

A ConcurrentListIterator watches concurrent add and remove changes on the list and moves the cursor logically so that items are neither skipped nor repeated.
