updateConcatenatedTransform

protected fun updateConcatenatedTransform(): Unit

Overrides UiComponentImpl.updateConcatenatedTransform

Updates this component's concatenatedTransform matrix, which is the parent's concatenatedTransform multiplied by this component's transform matrix.

Do not call this directly, use validateValidationFlags.CONCATENATED_TRANSFORM
