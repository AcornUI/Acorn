Properties

child

 val child: T

The child to add when the pop-up is activated. If the child has layoutData set, it is expected to be of type CanvasLayoutData

dispose

 var dispose: Boolean

If true, the pop-up will be disposed on removal.

focusFirst

 var focusFirst: Boolean

If true, when the pop-up is displayed, the first focusable element will be focused.

highlightFocused

 var highlightFocused: Boolean

If true and focusFirst is true, when the first focusable element is focused, it will also be highlighted.

isModal

 val isModal: Boolean

If true, a UI blocker will be added a layer below the child.

layoutData

 val layoutData: CanvasLayoutData

The layout data used for the pup-up's layout.

onCloseRequested

 val onCloseRequested: (child: T) -> Boolean

When a close is requested via clicking on the modal screen, the callback determines if the close succeeds. (true - remove the pop-up, false - halt removal)

onClosed

 val onClosed: (child: T) -> Unit

When this pop-up is removed, this callback will be invoked. If dispose is true, this callback will be called before disposal

priority

 val priority: Float

The greater the value the higher z-index the pop-up will receive.
