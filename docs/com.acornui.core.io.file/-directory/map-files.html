mapFiles

fun mapFiles(callback: (FileEntry) -> Boolean, maxDepth: Int = 100): Unit
Deprecated: use walkTopDown

Recursively invokes a callback on all descendant files in this directory.

Parameters

callback - Invoked once for each file. If the callback returns false, the iteration will immediately stop.

maxDepth - The maximum depth to traverse. A maxDepth of 0 will not follow any subdirectories of this directory.
